<h1 align="center">Sanft</h1>
一个基于React和TypeScript的UI组件库

___

* 引入

  ```react
  import { SanButton } from 'Sanft'
  ```

* 使用

  ```react
  <SanButton
      size={ButtonSize.Small}
      onClick={e => {
        e.preventDefault()
        message.info("hello")
      }}
      >
      Hello world
  </SanButton>
  ```

* 以storybook启动项目
  ```javascript
  npm run storybook
  ```
